Склонировать репозиторий
File-> New -> Project From Version Control -> Git
(в появившемся диалоговом окне указать ссылку на репозитоий)

VCS -> Git -> Clone

аналогично команде в консоле git clone ссылка_на_репозиторий

Последующая настройка проекта:
Папка с исходным кодом дб отмечена как source root
(правой кнопкой на имени проекта ->
mark directory as ->source root)

File -> Project Structure:
выбор sdk + указать папку для скомпилированных файлов

Получение изменений из удаленного репозитория:
VCS -> Git -> Pull

аналогично команде в консоле git pull

Индексация и фиксация изменений (коммит)
Красный цвет - отмечены новые файлы (непроиндексированные)
Голубой цвет - файлы с изменениями
Зеленый цвет - проиндексированные файлы

Правой кнопкой на файле или папке -> Git -> add -
для индексации изменений

аналогично команде в консоле git add ...

Правой кнопкой на файле или папке -> Git -> Commit File -
для создания коммита

аналогично команде в консоле git commit -m "message"

либо Commit & Push - для коммита и отправки изменений
в удаленный репозиторий

аналогично команде в консоле git push
