Книги в библиотеке:
Для каждой книги дб указаны:
автор
название
можно ли брать домой (или только в читальном зале)
isForHome

доступна для выдачи или нет
isAvailable

Книги хранятся в библиотеке
в библиотеку можно добавить одну книгу
в библиотеку можно добавить сразу несколько книг
должна быть возможность получить информацию по книге,
указав ее название
(получаемая информация: автор, название,
можно ли забрать домой, доступнали книга)
если книга не нйдена, сообщаем, что такой книги нет

должна быть возможность взять книгу на дом, указав название
должна быть возможность почитать книгу в читальном зале,
указав название
